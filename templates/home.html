About Me  

As a data science student (MSc) and Electronic Prescrbing and Medicines Administration (EPMA) Pharmacist, my professional interests lie within health data.

Electronic health records (EHR) systems, such as those that I work with, generate and store vast quantities of patient data that was typically underutilised within the NHS. Such data includes but is not limited to diagnoses, observations, lab results, measurements, patient demographics and prescribed medication.

Spurred by the impactful use of predictive analytics during the COVID-19 pandemic, I embarked on my journey into data science. Begining with SQL, improving my understanding of statistics, learning Python, delving into machine learning and much more. Now, I wish to use my skills to extract actionable insights from health datasets in professional environments.

This Website

This website was built using Python (Flask) and deployed to a Heroku server using Git. I also learnt rudimentary HTML and CSS in the process. The layout of this site relies on code provided by w3schools as their CSS template was adapted for my use.

The purpose of this website is to provide insight into my skills, experience and motivation as an aspiring
health data scientist. I hope this purpose is fulfilled. Please feel free to leave feedback here.
