Medtronic Task 1 Program

This program accepts CSVs as requested for task 1 of the tech test. It returns the pair of users with the lowest ecludian distance score.

Limitations and Assumptions
  • An assumption is made that the CSV will be in the exact format provided in the test.
  • This program assumes there are no duplicate records as the provided CSV did not contain any.
  • This program does not account for situations where multiple pairs have the best score.
